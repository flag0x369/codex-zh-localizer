# Roadmap

[简体中文](#路线图) | [English](#roadmap)

## 路线图

### 已支持

- 插件市场卡片中文化
- 插件详情页技能中文化
- 技能选择列表 `SKILL.md` / `agents/openai.yaml` 描述中文化
- Prompt 列表描述中文化
- 插件详情页应用连接器中文化
- dry-run、备份、恢复
- 临时 HOME smoke test
- 严格审计 `audit --strict`

### 近期计划

- 增加更多真实截图和中文化前后对比
- 将更多翻译规则和 skill 描述映射拆成可贡献的数据文件
- 增加翻译贡献指南和词汇表
- 准备 npm 发布，让安装命令变为 `npx codex-zh-localizer apply`
- 增加 release notes 模板

### 暂不计划

- 直接修改 `/Applications/Codex.app`
- patch `app.asar`
- 安装后台常驻进程
- 自动修改 shell 启动文件

## Roadmap

### Supported

- Plugin marketplace card localization
- Plugin detail skill localization
- Skill picker `SKILL.md` / `agents/openai.yaml` description localization
- Prompt list description localization
- Plugin detail app connector localization
- Dry-run, backup, and restore
- Temporary HOME smoke test
- Strict audit with `audit --strict`

### Near Term

- Add more real screenshots and before/after comparisons
- Move more translation rules and skill description mappings into contributor-friendly data files
- Add translation contribution guidance and glossary
- Prepare npm publishing so users can run `npx codex-zh-localizer apply`
- Add a release notes template

### Not Planned

- Directly modifying `/Applications/Codex.app`
- Patching `app.asar`
- Installing background services
- Modifying shell startup files automatically
